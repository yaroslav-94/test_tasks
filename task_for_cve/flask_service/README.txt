Описание сервиса

1) Необходимые глобальные переменные
    - UPLOAD_FOLDER - путь до сервиса, необходим, чтобы создать рядом папку для хранения изображений
    - ALLOWED_EXTENSIONS - разрешенные форматы изображений
    - RESCALE_SIZE - размер, к которому будут приводитьс все изображения
    - NEU_MODEL - пустой список, необходим для использования только одной модели за время работы сервиса
    - DEVICE - устройство, на котором будут происходить вычисления
    - PATH_MODEL - путь до сохраненной модели
    - DICT_BREEDS - словарь с классами, которые предсказывает модель, и значениеями из названий пород
    - SCORES_DICT - словарь для хранения предсказаний. Каждое предсказание для картинки расходует ресурсы, данная
    переменная поможет их немного сберечь


*) Интересное:
    - для повышения стабильности и качества предсказаний в данном проекте использован порог, идея которого - отсекать
    неуверенные предсказания на изображениях не относящихся к решаемой задаче. Например, в результате работы модели
    на картинке без каких-либо данных (noise.jpg) в результате преобразования softmax получается некоторый реальный
    класс. Данное поведение нежелательно. Для выяснения всех подробностей лучше попробовать
    "LIME - Local Interpretable Model-Agnostic Explanations" (https://github.com/marcotcr/lime)
